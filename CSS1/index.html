What si CSS?

  1. CSS stand for Cascading Style Sheets.
  2. CSS describe how HTML element to be displayed on screen,paper or in other media.
  3. Css not neccesary CSS saves a lot of work.It can control the layout of multiple web pages all at once.
  4. External stylesheets are stored in CSS files.

CSS can added to HTML document in 3 ways:

  • Inline - by using the style attribute inside HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS. is to keep the style in external CSS files. However, in this tutorial we will use inline and internal styles, because this is casier to demonstrate,and casier for you to try it yourself.

When comparing CSS class vs ID difference is the CSS class applies a style to multiple elements ,ID, on the other hand,appplies a style to one unique element,ID is also apecial in that you can use a spacial URL to link directly to an element and it's used by Javascript.

IN CSS, selectors are used to target a specific element or range of element on a web page. Once an element has been target,a style or set of style can be applied the element.

There is a wide rang of selectors available.Two of the most commonly used are class and ID. Both are used to target element to which a style should be applied.
